# MOEX Stock Tracker v2.0

## Описание
Скилл для получения информации об акциях на Московской бирже (MOEX). Позволяет узнать текущую цену, объём торгов и рассчитать недельный MACD для определения тренда.

## Как использовать
1. Скажите название компании на русском (например, «Газпром», «Сбер», «Яндекс»)
2. Скилл автоматически найдёт тикер через DuckDuckGo
3. Получите данные: цена в ₽, объём торгов, MACD-индикатор и сигнал тренда

## Примеры
- «Сколько стоят акции Газпрома?»
- «Покажи MOEX тикер Сбера»
- «Анализ акций Яндекса»

## Формат ответа
📊 Акции {НАЗВАНИЕ} ({ТИКЕР}):
💰 Цена: {LCURRENTPRICE} ₽
📊 Объём торгов: {VOLUME} шт.

📈 MACD (неделя):
• MACD-линия: {MACD_VALUE}

{'🟢 Бычий тренд' если MACD > 0, иначе '🔴 Медвежий тренд'}
