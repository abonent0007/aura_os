# Захват содержимого по URL

Захватывает текст веб-страниц по ссылкам. Поддерживает Twitter/X, Reddit и обычные сайты. Полезно когда пользователь делится ссылкой и хочет обсудить содержимое.

## Как использовать

Аура может вызвать этот скилл когда видит URL в сообщении. Или явно: «захвати содержание этой ссылки», «прочитай что там по ссылке».

## Поддерживаемые платформы

- 🐦 Twitter/X — через fxtwitter API (твиты + статьи)
- 📱 Reddit — через JSON API (посты + комментарии)
- 🌐 Обычные сайты — через HTML-парсинг

## Пример

> Юрий: «Что пишут по ссылке https://x.com/...»
> Аура: «[Захватывает содержимое] Автор пишет о...»
