# AURA Avatar — плагин визуальной анимации лица

## Назначение
Добавляет плавающее окно с анимированным женским лицом, которое синхронизирует движения рта с TTS-аудио. Работает как дополнение к AI-агенту AURA OS.

## Принцип работы
1. AI-агент получает текстовый ответ от LLM.
2. AI-агент передает текст в TTS-движок и одновременно в метод `speak()` этого плагина.
3. Плагин очищает текст от блоков кода и Markdown-разметки.
4. Плагин делает фонетический разбор: каждая буква → визема (положение рта).
5. Синхронно с TTS проигрывается анимация переключения PNG-изображений рта.
6. После окончания речи лицо переходит в idle-режим (глаза закрыты).

## Требования
- Python 3.10+
- Pillow (`pip install Pillow`)
- tkinter (встроен в Python на Windows/macOS; на Linux: `sudo apt install python3-tk`)

## Как прикрутить к AURA OS

### Способ 1: Прямой вызов из кода агента
```python
from aura_avatar.aura_avatar import AuraAvatar

avatar = AuraAvatar()

# Когда агент получил ответ от LLM:
llm_response = "Привет! Сегодня солнечно."

# Запускаем TTS и аватар одновременно
tts_engine.speak(llm_response)
avatar.speak(llm_response, audio_duration=tts_engine.get_duration(llm_response))

# Когда TTS закончил:
avatar.stop()

### Способ 2: Через хуки AURA OS
Если AURA OS поддерживает систему плагинов, добавьте plugin.json в директорию plugins/aura_avatar/.
Агент автоматически вызовет:

AuraAvatar.speak(text) при старте TTS

AuraAvatar.stop() при окончании TTS

AuraAvatar.shutdown() при выключении системы

### Особенности
Окно без рамок, всегда поверх других окон.

Автоматически игнорирует блоки кода (...) и инлайн-код (...).

Работает только с русским алфавитом (А-Я, а-я).

Не влияет на текст, который видит пользователь — только на анимацию.

### Добавление PNG-изображений
Изображения помещаются в папку assets/ после установки плагина.
Размер по умолчанию: 400×400 пикселей (настраивается в plugin.json).
Формат: PNG с прозрачностью (если рот накладывается слоем) или с фоном.
